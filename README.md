## Dependencies

Make sure you have ```opengl version>=3.3```, ```glfw``` and ```glm``` installed on you system.

## Build
### Linux
```git clone https://github.com/vigdail/opengl-terrain```

```cd opengl-terrain```

```mkdir build```

```cmake ../ && make```
